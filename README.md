### Модуль Kill Reason для Jailbreak Core (v1.0.0)

Добавляет систему указания причины убийства заключённых для CT-игроков на серверах Jailbreak. Позволяет фиксировать и отображать причину убийства в чате, а также при необходимости вызывать дополнительное действие (например, возрождение игрока).

---

#### Игровые возможности:
После убийства заключённого CT-игрок получает уведомление и доступ к меню выбора причины убийства.

* Автоматическое отслеживание убийств CT → T.
* Хранение списка последних убитых заключённых для каждого CT.
* Интерактивное меню выбора причины убийства.
* Отображение причины убийства в чате для всех игроков.
* Поддержка кастомных причин через конфигурационный файл.

---

#### Система причин убийства:
Каждая причина настраивается через конфиг и может включать:
* Текст причины, отображаемый в чате.
* Флаг `respawn`, позволяющий возрождать игрока при выборе причины.
* Гибкую систему добавления новых причин без перекомпиляции плагина.

---

#### Ограничения системы:
* Доступ к системе имеют только CT-игроки.
* Игрок должен быть жив на момент открытия меню.
* Причина может быть выбрана только для актуального списка последних убийств.

---

#### Файловая структура:
* `addons/configs/Jailbreak/killreason.ini` — конфигурация причин убийств и базовых настроек.
* `addons/translations/jailbreak_killreason.phrases.txt` — файл локализации (RU / EN / UA).

---

### Установка:
1. Распаковать содержимое в папку `game/csgo/addons/`.
2. Настроить причины убийств в `killreason.ini`.
3. Перезагрузить сервер или плагин.
4. Готово к использованию.
